## 快速启动

1. 克隆本仓库，或者 [下载源码压缩包](https://github.com/ElaBosak233/eco/archive/refs/heads/main.zip) 并解压
```bash
git clone https://github.com/ElaBosak233/eco.git
```

2. 使用 NPM/YARN 下载、安装依赖
```bash
npm install
```

3. 使用 NPM/YARN 运行
```bash
npm run start
```
